Project Title

SaaS Landing Page – Tailwind CSS Assignment

Project Description

This project is a modern and responsive SaaS landing page built using HTML5 and Tailwind CSS.
The design is inspired by the Genesis template and focuses on utility-first styling, layout structure, and responsive design without using custom CSS or JavaScript.

The goal of this assignment is to practice building professional web layouts using Tailwind’s utility classes while maintaining clean structure and visual consistency.

Features Implemented

Responsive Hero Section with headline, description, and call-to-action buttons

Use Cases / Features Section with multiple feature cards

Modern SaaS-style layout inspired by the Genesis template

Fully responsive design for mobile, tablet, and desktop

Built entirely with Tailwind CSS utility classes

Clean typography hierarchy and spacing system

Subtle hover effects and visual polish
